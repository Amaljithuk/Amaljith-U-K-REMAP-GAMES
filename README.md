# Amaljith-U-K-REMAP-GAMES

A collection of Python-based games with remapped controls for enhanced gameplay experience.

## 📋 Overview

This repository contains a series of games developed in Python, featuring customizable key remapping functionality to provide players with flexible and personalized control schemes.

## 🎮 Features

- **Python-based Games**: All games are built using Python
-
- **Easy Setup**: Simple installation process with all dependencies included

## 🚀 Getting Started

### Prerequisites
- Python 3.10.x installed on your system
- Dependencies as specified in the setup files

### Installation & Setup

1. **Download the game setup files** from this link:
   - 🔗 [Game Setup Files - Google Drive](https://drive.google.com/drive/folders/1X_tYFvj1lmk4bwOwBmiOCpGtlgy4FOJ3?usp=drive_link)

2. **Extract the setup files** to your desired directory

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run a game**:
   ```bash
   python game_name.py
   ```
